# 实施计划

## 已确认需求

- 服务名：贤得慌又云里雾里
- 运行环境：当前 OpenClaw 所在主机
- 对外访问：公网 IP + 端口
- 技术栈：由我拍板
- 权限模式：简单模式，保留进阶接口
- 分享：支持可选密码
- 上传限制：管理员不限，普通用户单文件 100MB
- 需要：文件预览、文件夹、搜索排序、准生产级管理后台

## 当前技术决策

- 前端/服务端一体：Next.js App Router + TypeScript
- 样式：Tailwind CSS + 玻璃拟态高对比深色 UI
- 数据层：Prisma
- 当前数据库：SQLite（便于现环境直接启动）
- 鉴权：自建 Session + HttpOnly Cookie
- 密码哈希：Argon2
- 存储：本地文件系统
- 分享：token + 可选密码 + 过期时间 + 最大下载次数

## 近期执行顺序

1. 搭建项目骨架与数据模型
2. 落登录鉴权与初始管理员
3. 落管理后台、文件页、分享页原型
4. 接入真实上传/下载/预览
5. 接入用户管理与文件夹管理
6. 做公网访问与进程守护
7. 视环境补 Docker/Nginx/Caddy

## 准生产目标

- 0.0.0.0 监听
- 可经主机公网 IP 访问
- 独立数据目录
- 可重启恢复
- 初始密码提示变更
- 安全头与最小权限文件访问
